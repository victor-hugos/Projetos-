# Backlog Produto

\
Módulos

\
* Coletores
  * Módulo para gerenciar os coletores que são responsáveis em conectar em fontes de dados internas e sincronizar os dados para o Datalike do ZELLO Connect
* Tranformadores
  * Módulo para gerenciar os transformadores que serão pipelines internas do Datalike que faz o tratamentos dos dados que estão espalhados em coleções e construem uma unica coleção com dados enriquecidos
* Observabilidade de Dados
  * Dashbord com numeros de tamanho de dados, coleções, acessos e outras métricas que sejam importantes para acompanhamento
* Catálogo de Dados
  * Portal de documentação das coleções do Datalike com seus Dicionário de dados
* Gerenciador de Chaves de Acesso
  * Módulo para gerenciar as API Keys que são responáveis do controle de acesso aos dados do Datalike
