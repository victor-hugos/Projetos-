# Apontamentos TCU

Aqui constam necessidades de usuários

| Dever ter no LOGSUS | Solicitante |
|----|----|
| O Ministério da Saúde deverá construir uma estrutura que possibilite o monitoramento transversalmente de todos as aquisições, armazenamento, logística e distribuição de insumos e medicamentos para o SUS. |    |
| A nova estrutura deve receber o andamento de aquisições para planejar armazenamento |    |
| A nova estrutura deve monitorar os insumos desde a aquisição até o usuário, de forma clara e transparente. |    |
| Deve ter integração com as compras feitas fora do Brasil, para monitorar o andamento no desembaraço. |    |
| Deve ter informações de compras em planejamento, desde o nascimento do DFD no PGC |    |
| O Ministério da Saúde deve ter controle de insumos da UBS de todo o Brasil |    |
|    |    |
