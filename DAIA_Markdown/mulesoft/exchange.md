# O que é

* Uma biblioteca de ativos
* O repositório central que é fundamental para o sucesso da construção de uma rede de aplicativos
* Fornece um local onde os ativos podem ser publicados para descoberta e reutilização

\
# O que o Exchange contém?

* Ativos públicos fornecidos pela Mulesoft e disponíveis em todas as contas para todos os usuários
* Conteúdo provado disponível apenas para pessoas em sua organização
  * Os ativos adicionados por qualquer pessoa em sua organização são adicionados ao seu exchange privado
  * A organização deve preenche-lo para conter tudo o que você precisa para criar seus projetos de integração
  * Inclui:
    * APIs
    * Conectores
    * Diagramas
    * Vídeos
    * Links
    * e etc
