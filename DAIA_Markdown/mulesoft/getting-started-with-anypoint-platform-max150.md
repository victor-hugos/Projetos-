# Objetivo do curso

Ao final do curso você deve ser capaz de:


1. Explicar como cada componente da plataforma Anypoint é usada para construir aplicações
2. Utilizar a plataforma Anypoint como:

   
   1. Repositório central de assets
   2. Construir aplicações que consomem assets e conecta sistemas
   3. Passar pelo ciclo de vida completo de uma API
