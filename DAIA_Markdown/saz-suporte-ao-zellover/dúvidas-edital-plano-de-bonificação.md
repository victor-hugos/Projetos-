# Dúvidas - Edital Plano de Bonificação


1. Para as funções que envolvem meta mensal, em virtude de se ter uma meta mensal, haverá necessariamente uma apuração mensal ou apenas do trimestre (ou seja, soma dos valores apurados no trimestre e divide por 3 para ter média mensal)? 

   \
   ***Resposta:*** *Será por TRIMESTRE. Importante ter um valor de referencia mensal para acompanhamento, contudo o fechamento será trimestral.*

   \
2. Para as funções que envolvem META MENSAL, o marco das OS’s dentro do Redmine, será a data da emissão do TRD? 

   \
   ***Resposta:*** *o marco será Status da Emissão da NFe.*

   \
3. Para a equipe, devemos levar em consideração a emissão dos TRD’s de cada Sprint? 

   \
   ***Resposta:*** *Sim.*

   \
4. Poderá haver alguma flexibilização para qualificação do bônus? 

   \
   ***Resposta:*** *Não. As regras estão definidas. Caso haja alguma alteração no edital, será divulgado a todos.*
